# 🔫 Revolver Roulette

An **offline** 6-shot Russian-roulette game built with Flutter. No backend, no network — just you, a side-view revolver, and your luck.

**Play on the web:** https://doonghwi.github.io/revolver-roulette/

## How to play
- The revolver holds **6 chambers**; one holds a **live round** (random each game).
- **Tap anywhere** to pull the trigger — the highlighted indicator shows where.
- **Survive** → the cylinder spins (sound + animation), and your *shots fired* counter ticks up.
- **Live round** → 💥 **gunshot, the screen flashes red, and the gun reloads** (new random chamber).

## Features
- 🎨 Side-view revolver drawn entirely with `CustomPaint` (no external art, copyright-free), muzzle pointing bottom-right.
- 📱 **Landscape-first** layout.
- 🔢 Horizontal shots-fired counter (`●●○○○○`).
- 👆 Pulsing tap indicator over the trigger.
- 🔊 **Sound effects** — cylinder spin + gunshot, synthesized from scratch (see `tool/make_sounds.py`), copyright-free.

## Tech
- Flutter (Android + Web)
- [`audioplayers`](https://pub.dev/packages/audioplayers) for sound
- Sounds & icon generated by the scripts in `tool/` (pure Python, no copyrighted assets)

## Build
```bash
flutter pub get
flutter run                       # Android / emulator
flutter build web --pwa-strategy=none --base-href /revolver-roulette/
flutter build apk --release
```

## Regenerate assets
```bash
python tool/make_sounds.py        # assets/sounds/*.wav
python tool/make_icon.py          # assets/icon/icon.png
dart run flutter_launcher_icons   # launcher icons
```
